#Implemented methods:

##v1.get\_sources
Если параметров нет - возвращает все h5 файлы. Если параметр есть - возвращает 
список файлов только с данным "reference" или ошибку если ни один файл не найден  

**params**: ref-строка
>{"reference": "ASDF"}

**result**: список h5 файлов (без расширения!) 
>["f1", "f1", ..., "fn"]

**error**:
>-32002, "Invalid reference" 


##v1.get\_source\_metadata
Возвращает метаданные по имени h5 файла.
Если имя неправильное возвращает ошибку

**params**: имя h5 файла (без расширения!)
>{"example": "f1"}

**result**: метаданные 

**error**:
>32003, "Invalid source"
